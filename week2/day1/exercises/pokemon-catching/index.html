Instructions

Open the script.js and Console part of this repository and write the code following these instructions

Iteration 1: Pokemon

Modify the Pokemon class by following the instructions below:

Class properties

  • should receive two arguments (one for the name and one for the type of the new created pokemons)
  • should have a property name. The value will come from the 1st argument
  • should have a property type. The value will come from the 2st argument

    • The attack() method

  • should be a function
  • It should have no parameters
  • should return the following string: "{NAME} uses {TYPE} attack" using the class properties

    • Iteration 2: Trainer

    Modify the Trainer class by following the instructions below:

    Class properties

  • should receive one argument (for the name of the created trainers)
  • should have a property name. The value will come from the argument
  • should have a property called ownedPokemons with a starting value of an empty array => []

    • The catchPokemon() method

  • should be a function
  • should have one parameter (The pokemon object to be catched when the method is called)
  • should use the parameter and add it to the array property of the class ownedPokemons
  • This method doesn't need a return value, it's only action will be updating the array when called

    • Iteration 3: Creating new objects based on the classes

    The trainer

  • use the variable trainer and assign a new object of the Trainer class
  • don't forget to give your trainer a name as it's argument!
  • Example - name: "Ash"

    • The pokemons

  • use the variable pokemon1, pokemon2 and pokemon3 and assign a new object of the Pokemon class to each one
  • don't forget to give your pokemon a name and type as it's arguments!
  • examples:
  • name: "Bulbasaur" type: "grass"
  • name: "Squirtle" type: "water"
  • name: "Charmander" type: "fire"

    • Catching some pokemons

  • invoke the catchPokemon method of the trainer object to catch the three created pokemon objects:
  • pokemon1, pokemon2 and pokemon3
  • remember, the method takes the pokemon to add as an argument!
  • this should be done in three separate lines

    • Once done, save your file and you should have the following output

          *** TEST 1. Should print the name of trainer and amount of pokemons (3) ***
      Ash has caught 3 Pokemons!
      *** TEST 2. Should print each of the three pokemon attacking ***
      Bulbasaur uses grass attack
      Charmander uses fire attack
      Squirtle uses water attack

    The output result could vary depending on how you named your trainer and your pokemon
