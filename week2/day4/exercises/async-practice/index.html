Open the script.js and Console part of this repository and write the code following these instructions

All the instructions are in the script.js itself

Here is the syntax for the setTimeout and setInterval functions:

            
      const timeoutId = setTimeout(() => {

        // any code to execute after the time below

      }, timeInMiliseconds)

      const intervalId = setInterval(() => {

        // any code to execute every interval determined by the time below

      }, timeInMiliseconds)


      // to clear them:
      clearTimeout( timeoutId )
      clearInterval( intervalId )
