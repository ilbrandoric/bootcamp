Instructions

Open the script.js and Console part of this repository and write the code following these instructions

Iteration 1

Assign a new value to the following variables

  • myFirstName with your first name (all lowercase).
  • myLastName with your last name (all lowercase).

To test, save the file with ctrl+S or command+S and the console log should print both variables

Iteration 2

Use the variable myFirstNameUppercased to store the value of myFirstName all in uppercase

Note: Don't write again the string with your name, use the variablemyFirstName

To test, save the file with ctrl+S or command+S and the console log should print your first name uppercased as myFirstNameUppercased

Iteration 3

Use the variable myLastNameCapitalized to store the value of myLastName capitalized (only the first letter as uppercase)

Note: Don't write again the string with your name, use the variablemyLastName

To test, save the file with ctrl+S or command+S and the console log should print your first name capitalized as myLastNameCapitalized

Iteration 4

Use the variable myFullName to store the value of myFirstNameUppercased and myLastNameCapitalized separated by a space

Click Run, the console log should print your full name as taken from both variables

To test, save the file with ctrl+S or command+S and the console log should print your first and last name, example: "BOB Macintosh"

Bonus

Use the variable myInitials to store the value of the first letter of your first name and the first letter of your last name

Note: User only the variable myFullName to achieve this

Example: if fullName is "BOB Macintosh" then the variable should be: "B.M.".

Hint: for Bonus: you can try using indexOf to identify the position of the empty space between the first name and last name.

To test, save the file with ctrl+S or command+S and the console log should print your initials, example: "B.M."
