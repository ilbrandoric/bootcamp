Instructions

Open the script.js and Console part of this repository and write the code following these instructions

Iteration 1

Create two constant variables

  • One called name with a string value: "Bob"
  • One called language with a string value: "english".

Iteration 2

Create a conditional that, depending on the value of language, stores a new string value in the variable message

  • For language "english" the value of message should be: "Hello [NAME]".
  • For language "spanish" the value of message should be: "Hola [NAME]".
  • For language "french" the value of message should be: "Bonjour [NAME]".
  • For any other value, the value of message should be: "I am not sure what are you trying to do, sorry! üç∫"

Save the file with ctrl+S or command+S to test your code.

  1. The Console should say: "Hello Bob"
  2. Manually Change the language variable to "spanish", now the Console should say: "Hola Bob"
  3. Manually Change the name variable name to "Patrick", now the Console should say: "Hola Patrick"
  4. Manually Change the language variable to "french", now the Console should say: "Bonjour Patrick"
  5. Manually Change the language variable to "potato", now the Console should say: "I am not sure what are you trying to do, sorry! üç∫"
